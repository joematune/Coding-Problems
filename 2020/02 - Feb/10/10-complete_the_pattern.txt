Task:

    You have to write a function pattern which returns the following
    Pattern(See Pattern & Examples) upto n number of rows.

    If n < 1 then it should return "" i.e. empty string.
    There are no whitespaces in the pattern.

Pattern:

    pattern(n):
    1
    22
    333
    ....
    .....
    nnnnnn
